# My-Poems-in-mongolian



(1)
тэгш хэм

Чи бяцхан гол мэт
миний зүрх рүү урсана.
Харин чиний зүрхний гүнд
бас нэгэн гол шивнэн урсана.
Эцэст нь би ойлгосон:
хоёр гол биш ажээ,
нэг л урсгал
хоёр зүрхээр дайран өнгөрдөг юм байна.



(2)
Навчны санамж

Миний нэгэн хэсэг
олон галактикийн дундуур аялсан гэрэл байжээ.
Эцэст нь дэлхийд ирээд
бяцхан навч болсон байна.
Тэр навчин дотроос
өөрийгөө хэрхэн олох вэ гэж
би удаан бодсон.
Харин одоо ойлгож байна:
би навчийг хайгаагүй,
навч л намайг
санаж байжээ.
